# 📊 E-Commerce Sales Dashboard

## 📌 Overview

This project focuses on designing an interactive dashboard to analyze e-commerce sales data and present key business insights in a clear and visually engaging way.

---

## 🎯 Objective

* Track sales performance
* Analyze revenue trends
* Identify top-performing products
* Enable easy data exploration

---

## 🧰 Tools Used

* Power BI

---

## 📊 Dashboard Features

* KPI cards for total sales, revenue, and performance
* Sales trend analysis over time
* Product and category performance insights
* Interactive filters and slicers
* Clean and user-friendly design

---

## 💡 Key Insights

* Identifies top-selling products
* Highlights sales trends and patterns
* Supports data-driven decision making

---

## 🎨 Design Focus

* Simple and intuitive layout
* Clear visual hierarchy
* Business-friendly labels and charts
